# Виселица
Игра "Виселица" с консольным интерфейсом. Правила игры по [ссылке](https://ru.wikipedia.org/wiki/%D0%92%D0%B8%D1%81%D0%B5%D0%BB%D0%B8%D1%86%D0%B0_%28%D0%B8%D0%B3%D1%80%D0%B0%29).
## Установка и запуск
1. Клонируйте репозиторий
```
git clone https://github.com/ibobrov/hangman.git
```
2. Перейдите в директорию
```
cd hangman
```
3. Запустите игру
```
python main.py
```
## Пример использования
При запуске приложение предлагает либо начать новую игру, либо выйти.
```
Начать новую игру или выйти? (newgame/exit) 


```
При начале новой игры начинается процесс угадывания слова:
```
  _________
  |/      |
  |       
  |       
  |       
  |       
__|_______
_____
Всего ошибок: 0. Последний ввод: -.
Введите букву русского алфавита: 
```
Пользователь вводит "а"
```
  _________
  |/      |
  |       
  |       
  |       
  |       
__|_______
_а__а
Всего ошибок: 0. Последний ввод: а.
Введите букву русского алфавита: 
```
Пользователь вводит "ц"
```
  _________
  |/      |
  |       O
  |       
  |       
  |       
__|_______
_а__а
Всего ошибок: 1. Последний ввод: ц.
Введите букву русского алфавита: 
Такой буквы нет в загаданном слове.
```
По завершении игры выводится результат (выигрыш или проигрыш):
```
  _________
  |/      |
  |       O
  |      /|\
  |      / \
  |       
__|_______
_а__а
Всего ошибок: 6. Последний ввод: й.
Введите букву русского алфавита: 
Вы проиграли! Правильное слово: удача
```
Возвращаемся к выбору :
```
Начать новую игру или выйти? (newgame/exit) 


```
## Структура проекта
- `main.py` - основной файл с логикой игры
- `words.txt` - содержит слова, выбираемые в игре
- `hangman_stages.json` - содержит <<этапы>> - рисунки виселицы на различных стадиях
- `.pre-commit-config,yaml` - конфигурация для pre-commit хука
- `.gitignore` - исключения для git
